<h1> AndroidWatch </h1>

<p>This project aims to be a clone  of android watch</p>

<h2> Design guide </h2> 

<h3>Alarm Screen</h3>

<p>In this screen the user should be able to add a new card in the + button</p>
<p>This card will define the hour of the alarm. More details about the card in the components section.</p>

<img src="assets/readmeAssets/Screenshot_20220206-133541.png" alt="MarineGEO circle logo" style="height: 700px; "/>

<h3>Clock Screen</h3>
<p>In this screen the user should be able to see the hour in different timezones.</p>
<p>In the + button the user will add a new card that should display the hour in the especifield timezone.</p>

<img src="assets/readmeAssets/Screenshot_20220206-133608.png" alt="MarineGEO circle logo" style="height: 700px; "/>
<h3>Timer Screen</h3>
<p>In this screen the user should be able to set a timer.</p>
<p>The user will see the progression of the timer in the timerComponent, that should behave like a progress bar.</p>
<p>he user shoud be able to enable a sound to play when the counter sets to 0.</p>
<img src="assets/readmeAssets/Screenshot_20220206-133620.png" alt="MarineGEO circle logo" style="height: 700px; "/>
<h3>StopWatch Screen</h3>

<p>In this screen the user should be able to start a stopwatch.</p>
<p>The user sould be able to see the time passing, make turns, pause and stop the timer.</p>
